# GOLDEN GAME

スロットの出目で収入を得ながら、家族シンボルを増やして支払いを乗り越えるブラウザゲームのプロトタイプです。

## コンセプト

- スピンして15個のシンボルを出す
- シンボルごとの効果で収入を計算する
- ターンごとに新しいシンボルを1つ選んでデッキに追加する
- 指定ターンごとに支払いが発生する
- 支払いに失敗するとゲームオーバー

## 遊び方

1. `index.html` をブラウザで開く
2. 「スピン」を押す
3. 獲得コインを確認する
4. 3つの候補から家族シンボルを1つ選ぶ
5. 支払い期限までに必要なコインを集める
